# coin-feedback

Coin Feedback is a SwiftUI iOS app for capturing, organizing, and sharing product feedback. It focuses on a clean, glass-inspired UI, simple entry flows, and optional paywall support for premium features.

## Motivation

This app was motivated by a LinkedIn article I created: https://www.linkedin.com/posts/mauricioneri_ai-softwareengineering-indiedev-activity-7422308237443616768-IUl-?utm_source=share&utm_medium=member_desktop&rcm=ACoAAACdINQBDatdYbVlZbpVv0tX_0WwRxWLaQE

## Highlights

- Create, edit, and review feedback entries
- Elegant SwiftUI UI components and theming
- Email sharing support
- Optional paywall and in-app purchase flow
